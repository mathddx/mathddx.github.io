Ao invés de fazer relatórios e estudar para as provas, o divertido foi fazer coisas loucas aqui nesse site. Em especial, conseguir colocar ele pra rodar no Thinkpad, e pra isso foi feito o seguinte:
- Comecei seguindo a instalação do ruby e do [Jekyll](https://jekyllrb.com/docs/installation/ubuntu/), e foi bem tranquilo sem estresse. O que deu um certo trabalho foi logar no github:

```sh
sudo pacman -S github-cli
```
Além disso, uns erros aconteceram ao rodar o comando bundle pois haviam erros ao instalar as coisas a partir desse comando:

```sh
gem install jekyll bundler
```

A solução, foi simplesmente adicionar o PATH no ~/.bashrc, a partir das seguintes linhas:

```sh
export GEM_HOME=$HOME/gems
export PATH=$HOME/gems/bin:$PATH
```

No mais, os seguintes comandos são usados:

### Iniciar site local:
```sh
bundle exec jekyll main
```
#### Colocar no github:
```sh
 git add .
 git commit -m 'comentário'
 git push origin main

```
